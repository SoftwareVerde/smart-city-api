Parking Meters

get

Get a single parking meter.

Accessing the API
URL /api/v1/parking-meters?get=1
Inputs (Query String)
id : number The ID (primary key) used for this API. Mutually exclusive from meter_id. (Optional)
meter_id : string The Columbus-Assigned ID. Mutually exclusive from id. (Optional)
Outputs (Json)
id : number The ID (primary key) used for this API. Mutually exclusive from meter_id.
meterId : string The Columbus-Assigned ID. Mutually exclusive from id.
location : string The nearest address to the meter. May not have the street number.
maxDwellDuration : number The maximum number of minutes the meter can be loaded for.
isHandicap : boolean 'true' if the meter corresponds to a handicap parking space. 'false' otherwise.
rate : number The rate in dollars and cents (e.g. 2.75) that would be charged for a full meter (see maxDwellDuration).
isChargingStation : boolean 'true' if the meter is also a car charging station. 'false' otherwise.
latitude : number The latitude of the meter (may correspond to the nearest address, not the meter itself).
longitude : number The latitude of the meter (may correspond to the nearest address, not the meter itself).

list

Get all parking meters.

Accessing the API
URL /api/v1/parking-meters?list=1
Inputs (Query String)

None.

Outputs (Json)
parkingMeters : array An array of parking meter objects, as defined in the 'get' response above.

search

Get all parking meters matching specific criteria.

Accessing the API
URL /api/v1/parking-meters?search=1
Inputs (Query String)
radius : number Radius (in meters) around the below latitude and longitude about which to return results (inclusive). (Optional)
latitude : number The latitude to be used as the center for radius-based searching. (Optional)
longitude : number The longitude to be used as the center for radius-based searching. (Optional)
street : string Return parking meters with the given street name. Can include wildcards (via '%'). (Optional)
max_dwell_duration_greater_than : number Return parking meters whose max dwell duration is than greater than this amount (in minutes) (inclusive). (Optional)
max_dwell_duration_less_than : number Return parking meters whose max dwell duration is than less than this amount (in minutes) (inclusive). (Optional)
rate_greater_than : number Return parking meters whose full rate is greater than this amount (in dollars) (inclusive). (Optional)
rate_less_than : number Return parking meters whose full rate is less than this amount (in dollars) (inclusive). (Optional)
is_handicap : boolean Return parking meters designated as a handicap when set to 1 or not handicap when set to 0. (Optional)
is_charging_station : boolean Return parking meters designated as a charging station when set to 1 or not charging station when set to 0. (Optional)
Outputs (Json)
parkingMeters : array An array of parking meter objects, as defined in the 'get' response above.
