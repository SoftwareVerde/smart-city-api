Parking Meters

get

Get a single parking meter.

Accessing the API
URL /api/v1/parking-meters?get=1
Inputs (Query String)
id The ID (primary key) used for this API. Mutually exclusive from meter_id.
meter_id The Columbus-Assigned ID. Mutually exclusive from id.
Outputs (Json)
id The ID (primary key) used for this API. Mutually exclusive from meter_id.
meterId The Columbus-Assigned ID. Mutually exclusive from id.
location The nearest address to the meter. May not have the street number.
maxDwellDuration The maximum number of minutes the meter can be loaded for.
isHandicap 'true' if the meter corresponds to a handicap parking space. 'false' otherwise.
rate The rate in dollars and cents (e.g. 2.75) that would be charged for a full meter (see maxDwellDuration).
isChargingStation 'true' if the meter is also a car charging station. 'false' otherwise.
latitude The latitude of the meter (may correspond to the nearest address, not the meter itself).
longitude The latitude of the meter (may correspond to the nearest address, not the meter itself).

list

Get all parking meters.

search

Get all parking meters matching specific criteria.
