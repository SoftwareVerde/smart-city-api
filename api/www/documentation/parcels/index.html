Parcels

get

Get a single parcel.

Accessing the API
URL /api/v1/parcels?get=1
Inputs (Query String)
id : number The ID (primary key) used for this API. Mutually exclusive from parcel_id. (Optional)
parcel_id : string The Columbus-Assigned ID. Mutually exclusive from id. (Optional)
Outputs (Json)
id : number The ID (primary key) used for this API. Mutually exclusive from parcel.
parcelId : string The Columbus-Assigned ID. Mutually exclusive from id.
exemptPropertyAppraisalLandValue : number The appraisal land value for exempt properties only.
exemptPropertyAppraisalBuildingValue : number The appraisal building value for exempt properties only.
exemptPropertyAppraisalTotalValue : number The appraisal total value for exempt properties only.
taxablePropertyAppraisalLandValue : number The appraisal land value for taxable properties.
taxablePropertyAppraisalBuildingValue : number The appraisal building value for taxable properties.
taxablePropertyAppraisalTotalValue : number The appraisal total value for taxable properties.
auditorsMapNumber : string The auditor's map number with the first character being the auditor's map book code (AUDMPRE), the next three characters being the auditor's map book page (AUDPAGE), and the final five characters containing the auditor's map suffix codes (AUDMSUF).
auditorsRoutingSequence : string The auditor's routing sequence with the first three numbers being the routing sequence, followed by a period, then two numbers representing multiple card sequences.
appraisalLandUseCode : string The auditor's land use code for appraisal purposes (not to be used as zoning codes)
currentAgriculturalUseValue : string The current agricultural use value.
schoolDistrictCode : string The school district code.
containsHomesteadExemptionValues : boolean True if parcel contains homestead exemption values.
taxBillMailingAddressLine1 : string Line 1 of the tax bill mailing address.
taxBillMailingAddressLine2 : string Line 2 of the tax bill mailing address.
taxBillMailingAddressLine3 : string Line 3 of the tax bill mailing address.
taxBillMailingAddressLine4 : string Line 4 of the tax bill mailing address.
transferDate : date The date of the last ownership transfer.
transferYear : number The year of the last ownership transfer.
ownerName1 : string Name of the first owner of the parcel.
ownerName2 : string Name of the second owner of the parcel, if any.
ownerName3 : string Name of the third owner of the parcel, if any.
ownerAddress1 : string Line 1 of the address of the owner, usually the parcel address for residential properties.
ownerAddress2 : string Line 2 of the address of the owner, usually the parcel address for residential properties.
auditorsNeighborhoodCode : string The auditor's in-house code identifying the neighborhood of the parcel.
floodInfo : string Flood info.
propertyClass : string The parcel property class. Valid entries include:
  • C >> Commercial property
  • E >> Exempt property
  • F >> Agricultural property
  • I >> Industrial property
  • M >> Mineral
  • R >> Residential property
  • U >> Utility
numberOfPropertyCards : number The number of property cards used to record the parcel.
acreage : number The acreage of the parcel.
lastSalePrice : number The last known valid sale price.
totalTaxableValue : number The total taxable value for the parcel.
city : string The city the parcel is within.
state : string The state the parcel is within.
zipcode : string The zip code the parcel is within.
descriptionLine1 : string Line 1 of the parcel description.
descriptionLine2 : string Line 2 of the parcel description.
descriptionLine3 : string Line 3 of the parcel description.
taxDesignation : string The parcel's tax designation.
primaryBuildingAreaSquareFeet : number The area of the primary building on the parcel in square feet
dwellingType : string The dwelling type. Valid entries include:
  • 0 >> No dwelling
  • 1 >> Has dwelling
totalRoomCount : number The total number of rooms.
fullBathCount : number The number of full baths.
halfBathCount : number The number of half baths.
bedroomCount : number The number of bedrooms.
centralAirStyle : string The style of central air. Valid entries include:
  • 0 >> None
  • 1 >> Heat
  • 2 >> Heat & Air
buildingCondition : string The building condition. Valid entries include:
  • V >> Very Good
  • G >> Good
  • A >> Average
  • F >> Fair
  • P >> Poor
hasFireplaces : boolean True if the parcel contains fireplaces.
buildingGrade : string The grade of the building. Valid entries include:
  • A
  • B
  • C
  • D
  • E
height : string An indication of the height of the house. Can be viewed as a "style" of house.*
storyCount : number An indication of the number of stories in the house. Generally represented as a number of feet.*
yearBuilt : string The year the building was built. Valid entries include:
  • Any calendar year from 1920 to the present year.
  • "OLD" for buildings remodeled before 1920.
  • "E 99" where 99 is a valid calendar year for estimated year of remodel
propertyType : string The property type. Valid entries include:
  • 1 >> Vacant log
  • 2 >> Dwelling
  • 3 >> Commercial
  • 4 >> Industrial
  • 5 >> Other
wallCode : string The wall code. Valid entries include:
  • 1 >> Wood or aluminum
  • 2 >> Stucco
  • 3 >> Tile
  • 4 >> Concrete block
  • 5 >> Metal
  • 6 >> Concrete
  • 7 >> Brick
  • 8 >> Stone
  • 91 >> 5/6 frame with 1/6 masonry
  • 92 >> 4/6 frame with 2/6 masonry
  • 93 >> 3/6 frame with 3/6 masonry
  • 94 >> 2/6 frame with 4/6 masonry
  • 95 >> 1/6 frame with 5/6 masonry
latitude : number The latitude of the meter (may correspond to the nearest address, not the meter itself).
longitude : number The latitude of the meter (may correspond to the nearest address, not the meter itself).

* - Some of the details here have been inferred from the data and are not officially documented at the Franklin County Auditor's website.

list

Get all parcels.

Accessing the API
URL /api/v1/parcels?list=1
Inputs (Query String)

None

Outputs (Json)
parcels : array An array of parcel objects, as defined in the 'get' response above.

search

Get all parcels matching specific criteria.

Accessing the API
URL /api/v1/parcels?search=1
Inputs (Query String)
radius : number Radius (in meters) around the below latitude and longitude about which to return results (inclusive). (Optional)
latitude : number The latitude to be used as the center for radius-based searching. (Optional)
longitude : number The longitude to be used as the center for radius-based searching. (Optional)
address : string Match one of the owner address lines. Can include wildcards (via '%'). (Optional)
transfer_date_after : string Return parcels whose last transfer date is after the provided date (yyyy-MM-dd HH:mm:ss format) (inclusive). (Optional)
transfer_date_before : string Return parcels whose last transfer date is before the provided date (yyyy-MM-dd HH:mm:ss format) (inclusive). (Optional)
owner_name : string Match one of the owner names. Can include wildcards (via '%'). (inclusive). (Optional)
last_sale_price_greater_than : number Return parcels whose last sale price is greater than this amount (in dollars) (inclusive). (Optional)
last_sale_price_less_than : number Return parcels whose last sale price is less than this amount (in dollars) (inclusive). (Optional)
city : string Return parcels within the provided city. (Optional)
zipcode : string Match on parcel zip code. Can include wildcards (via '%'). (Optional)
acreage_greater_than : number Return parcels whose acreage is greater than this amount (in acres) (inclusive). (Optional)
acreage_less_than : number Return parcels whose acreage is less than this amount (in acres) (inclusive). (Optional)
primary_building_area_greater_than : number Return parcels whose primary building area is greater than this amount (in square feet) (inclusive). (Optional)
primary_building_area_less_than : number Return parcels whose primary building area is less than this amount (in square feet) (inclusive). (Optional)
Outputs (Json)
parcels : array An array of parcel objects, as defined in the 'get' response above.
